# 助贷CRM系统

#### 介绍
助贷CRM系统，不限坐席、支持总分公司架构使用、支持一键拨号、技术支持数据商、客户实时提醒、公海自动回收、黑名单限制、IP限制登录、支持上门拜访登记、支持业绩统计、一键清楚数据等等，欢迎体验。

使用JAVA语言开发，java1.8+mysql+redis+websocket

演示地址：
